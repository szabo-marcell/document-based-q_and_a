English:
This project is a document-based question-answering (Q&A) system. After being provided with relevant documents or text, the AI analyzes the information and generates answers to 
user questions based solely on the given content. It uses chain-of-thought reasoning and inference to understand context and produce accurate responses.The project was
developed using Claude Code and is intended to demonstrate its capabilities in building AI-powered applications.

Magyar:
A projekt egy dokumentumalapú kérdés-válasz alkalmazás. Miután a felhasználó megadta a szükséges dokumentum(ok)at vagy szöveget, az AI elemzi az információkat és választ 
szolgáltat a feltett kérdésre a megadott tartalom alapján. Gondolatláncot (chain_of_thought) és inference-t használ, hogy megértse a kontextust, és, hogy megfelelő választ
állítson elő. A projekt a Claude Code segítségével készült, egyik célja, hogy demonstrálja a Claude Code képességeit.
